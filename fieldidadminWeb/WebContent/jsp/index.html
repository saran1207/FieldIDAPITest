Template Info

Translucent Fluidity is a simple fluid template that can be used for anything you like. It has a fixed body background, a main container with a transparent background, and a simple horizontal tabbed menu with transparent backgrounds. You can make your own transparent backgrounds like I did from the tutorial at:
http://www.wannabegirl.org/translucent/.
I've tested the template in Firefox 1.5, IE6, and Opera 8.5. Enjoy!
