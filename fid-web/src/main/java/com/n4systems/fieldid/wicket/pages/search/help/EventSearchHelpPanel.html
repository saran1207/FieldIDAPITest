The search bar accepts a number of different parameters. Below is a list of the terms you can use. For example, location=building 21. You can combine search terms with AND, OR.

  • IDENTIFIERS

    • id
    • ref
    • rfid

  • OWNER

    • org
    • cu
    • div
    • owner (the lowest level)
    • assigned

  • DATES

    • duedate
    • completeddate

  • USERS

    • usercreated
    • usermodified
    • performedby
    • assignee

  • TYPE

    • at (asset type)
    • atg (asset type group)
    • et (event type)
    • etg (event type group)

  • OTHER

    • result
    • comments
    • notes
    • score

CUSTOM ATTRIBUTES

Custom attributes can be searched by all lowercase and no spaces. eg Working Load would be workingload=21 Tons
