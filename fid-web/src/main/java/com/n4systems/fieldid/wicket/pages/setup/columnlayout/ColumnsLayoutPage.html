Creating a Layout

To create a new report layout, simply expand a group (ie, Ownership) to select a new field. Click on + to add it to your report.

To add a custom field (ie Manufacturer or Size), expand a custom group and click Add New. Type the name of the attribute as it appears on the screen. Click Save. Click + to add it to your report.

Modifying a layout

Under Current Report Layout hold and drag a column to order the layout.

Use the default sorting to set how the results should be ordered.
