The search bar accepts a number of different parameters. Below is a list of the terms you can use. For example, location=building 21. You can combine search terms with AND, OR.

  • IDENTIFIERS

    • id
    • ref
    • rfid

  • OWNER

    • org
    • cu
    • div
    • owner (the lowest level)
    • assigned

  • DATES

    • assetcreated
    • assetmodified
    • assetidentified

  • USERS

    • usercreated
    • usermodified
    • identifiedby

  • TYPE

    • at (asset type)
    • atg (asset type group)

  • OTHER

    • comments
    • po
    • order
    • lasteventdate
